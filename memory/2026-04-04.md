## Session: abogadosdrc.com — Rediseño completo Angular → Next.js

### Contexto
Kicks pidió rediseño completo del sitio https://abogadosdrc.com (DRC Consultores Fiscales, Durango México).
Repo: https://github.com/kikindb/abogadosdrc
Branch de trabajo: `redesign/nextjs-modernization` → mergeado a `master`

### Decisiones arquitectónicas
- Migración de Angular 18 + Scully → **Next.js 15 App Router con `output: 'export'`** (static site)
- CSS: Bootstrap 4 + jQuery → **Tailwind CSS 4**
- Iconos: FontAwesome bundled → **Lucide React**
- Blog: Scully markdown → **gray-matter + marked**, posts en `content/blog/*.md`
- Firebase Hosting sigue siendo el deploy target, output en `out/`
- `firebase.json` actualizado: `"public": "out"`, sin rewrites (static export), con cache headers

### i18n
- Selector ES/EN en navbar con `useI18n()` hook y `I18nProvider` (client-side, localStorage)
- Traducciones en `lib/i18n.ts`
- Homepage es Server Component puro para SEO; el i18n es encapsulado en `HomeClient` (client component)
- Blog posts solo en español; Google Translate widget en blog posts únicamente (no en todas las páginas)
- `<meta name="google" content="notranslate">` en layout para evitar indexado de versiones traducidas

### SEO implementado
- Homepage: `LegalService` JSON-LD + `FAQPage` + `WebSite` schemas
- Blog posts: `Article` + `BreadcrumbList` JSON-LD, microdata `itemProp`
- Blog index: `ItemList` JSON-LD
- hreflang `es-MX` + `x-default` en todas las páginas
- Open Graph + Twitter Card completos por página
- `keywords` meta en todas las páginas
- `robots: { googleBot: { max-snippet: -1, max-image-preview: 'large' } }`
- `notranslate` meta para evitar contenido duplicado

### Accesibilidad (WCAG)
- Skip link "Saltar al contenido principal" en layout
- `aria-expanded`, `aria-controls`, `aria-current`, `aria-pressed` implementados
- `prefers-reduced-motion` en CSS
- `:focus-visible` ring visible (dorado 3px)
- Escape cierra menú mobile
- `aria-hidden` + `focusable="false"` en todos los SVGs decorativos
- `aria-label` en todos los links externos con "(abre en nueva pestaña)"
- `<address>` para info de contacto
- `role="group"` + `aria-label` en LangSwitcher

### Performance (Lighthouse)
- LCP fix: `fetchPriority="high"` + `<link rel=preload>` para mazo.webp
- Google Translate movido solo a blog/[slug]/page.tsx con `defer`
- `loading="lazy"` en logo footer e imágenes no críticas
- Cache headers en firebase.json:
  - `/_next/static/**` → 1 año immutable
  - `*.js, *.css, *.woff` → 1 año immutable
  - `/assets/img/**` → 1 día + 7 días stale-while-revalidate
  - HTML → `no-cache, must-revalidate`
- Security headers: `X-Content-Type-Options`, `X-Frame-Options: DENY`, `Referrer-Policy`
