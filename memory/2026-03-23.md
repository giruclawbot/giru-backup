## Lecciones aprendidas (sesión 2026-03-23)
- **Astro + base path**: `import.meta.env.BASE_URL` incluye trailing slash (`/nombres/`). Para concatenar manualmente usar `.replace(/\/$/, '')`. Para assets, usar `new URL('file', Astro.site)` o la forma `BASE_URL` con cuidado.
- **Grid responsive**: usar `minmax(min(280px, 100%), 1fr)` en vez de `minmax(280px, 1fr)` para evitar overflow en mobile.
- **Embedded git repos**: al copiar skills con `.git/` interno, usar `git rm --cached -f` y re-copiar solo los archivos necesarios.
- **No decir "terminé" hasta verificar**: siempre correr build + curl en preview antes de reportar como completo.
